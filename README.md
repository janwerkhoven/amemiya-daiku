# Amemiya Daiku

## What
Homepage of a traditional Japanese carpenter (daiku) named Kunihiro Amemiya (雨宮国広) (www.amemiya-daiku.jp)

Features:
* Localization to both English and Japanese
* Minimalistic design
* Mobile responsive
