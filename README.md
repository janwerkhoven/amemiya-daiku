# Amemiya Daiku

## What
Homepage of a traditional Japanese carpenter (daiku) named Kunihiro Amemiya (雨宮国広) (www.amemiya-daiku.jp)

## Features
* Localization to Japanese and English
* 100% precompilation for lightweight load and SEO boost
* Minimalistic design
* Mobile responsive

## How
* Node
* Grunt
* Handlebars
* HTML5
* SASS
* jQuery
* Photoshop
* Illustrator
